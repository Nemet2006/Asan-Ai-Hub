# UI Layout and Brand Direction

## Vizual istiqamet

- Dovlet-tech ve etibarli ton
- Acıq, hava dolu card-based layout
- Teal, derin lacivert ve isti narinci aksentler
- Serif basliqlar, sade sans-serif metn
- Glassmorphism yox, yumsaq satha malik idarəetmə panelləri

## Reng sistemi

- Primary: `#0f6e8c`
- Deep navy: `#0a2942`
- Support green: `#147d64`
- Accent orange: `#d97828`
- Background: `#f3f7f5`
- Text: `#10253a`

## Core UI pattern-lari

- Sticky navigation shell
- Boyuk editorial hero
- KPI kartlari
- Modul esasli feature grid
- Split layout ile dashboard preview
- Sol sidebar + sag emeliyyat kontenti olan dashboard strukturu

## Ana sehife axini

1. Hero
   - Basliq: "Muracieti sekilden anlayan ve icra neticesini yoxlayan vahid platforma"
   - CTA: vetendas axini ve API memarligi
2. Is prinsipi
3. Platforma imkanlari
4. AI qerar masasi preview-si
5. Ekosistem bloklari
6. Deyer kartlari
7. Pilot merheleleri
8. Sened paketi

## Dashboard pattern-i

- Sol sidebar
- Yuxarida status/summary hero
- KPI card row
- Asagida 2 kolonlu emeliyyat bloklari

Bu pattern vetendas, qurum ve nezaret panellerinde eyni qalir, amma kontent her rol ucun deyisir.
